A Primer in Econometric Theory

This page collects code in the form of live Jupyter notebooks for the text A Primer in Econometric Theory. The code is organized into directories according to the chapter of the text in which that code or figure appears.

Instructions:

* To run code in a notebook after opening it from the links below type Shift+Enter. This will execute the current active cell and shift focus to the next cell. Alternatively you can run all cells in a given notebook sequentially from the Cell menu item.

* You can edit code live in the browser but any changes will be lost after you end the session. To make permanent changes you need to download the notebooks and run them locally. Information on how to do this is available from the text home page
